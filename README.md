# fetch-slack-emoji

## Python environment

Set up the virtual environment:

``` sh
pdm py install cpython@3.12
pdm use -f cpython@3.12
```

Install dependencies:

``` sh
pdm install
```

## Git hooks

``` sh
pdm run pre-commit install
```
